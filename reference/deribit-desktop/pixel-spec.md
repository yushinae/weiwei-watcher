## Deribit 期权链（桌面端）像素规格草案（基于截图）

> 说明：Deribit 页面无法稳定抓取 computed styles，因此以下为“截图量化 + 复刻用 token”。
> 后续会通过“截图对比 + 像素 diff”继续收敛到 100%。

### 基础色板
- 背景主色：#0B0C0E（近黑，略带灰）
- 表头底色：#111315
- 行背景（偶/奇）：#14171A / #15191C
- 网格线：rgba(255,255,255,0.08)（细线、低对比）
- 文字主色：rgba(255,255,255,0.92)
- 文字次级：rgba(255,255,255,0.55)

### 强调色
- UI 蓝（按钮/选中）：#2F6BFF
- Strike/Spot 紫（spot pill、紫色带）：#9B4DFF
- Up：#35D07F
- Down：#FF5C74

### 版式（桌面端）
- 顶部全局导航：54px 高，背景黑 + 轻微 blur
- 二级工具条按钮：36px 高，圆角约 10px，边框 rgba(255,255,255,0.09)
- 表格行高：32px
- 单元格左右 padding：7px（紧凑密度）
- 字号（表格主数字）：12px
- Mark cell：主值 12.5px，IV 10.5px（上下两行）
- Strike：主值 ~13-14px（白），Dist% ~10-11px（红/绿）

### 关键视觉特征
- hover：整体微亮（brightness ~1.06），不出现夸张 glow
- 选中行：蓝色半透明铺底（覆盖 call+put 两侧）
- Spot 线：细线（1px），颜色 = Spot 紫
- Spot pill：紫底白字，轻投影（不是霓虹外发光）
- Expected move 带：在 strike 列范围内的竖向淡紫半透明区域（可带左右边线）
